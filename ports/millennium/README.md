# Deck Shelves Millennium

This package is the Millennium port of Jonathan Santos's
[Deck Shelves](https://github.com/santojon/Deck-Shelves) plugin for Decky
Loader. It targets Steam Big Picture on Millennium and uses a Lua backend.

## Install

Install the extracted `deck-shelves` directory in Millennium's plugins
directory, enable the plugin in Millennium, and restart or reload Steam.

This package intentionally contains only the runtime payload. User settings
and backups are created beside the installed plugin at runtime and are never
included in release archives.

## Compatibility

The exact upstream commit, port version, minimum Millennium version, and
submodule revisions are recorded in `port/upstream.json` inside the package.
See the repository's `docs/port/` directory for the supported feature matrix,
development setup, testing procedure, and upstream-update workflow.
