# Security Policy

## Supported versions

The latest Deck Shelves Millennium release and the current `main` branch
receive security fixes.

## Report a vulnerability

Please use a private
[GitHub security advisory](https://github.com/DevsNate/DeckShelves-Millennium/security/advisories/new)
instead of opening a public issue. Include reproduction steps, affected
versions, impact, and any suggested mitigation.

## Scope

Security reports may cover:

- Millennium bootstrap and Lua backend behavior
- Settings and file-path handling on Windows
- Steam webview or Big Picture UI injection
- Network requests and update handling
- Release packaging and dependency integrity

Reports caused exclusively by unsupported third-party modifications or without
a practical security impact may be closed as out of scope.

Deck Shelves Millennium is an independently maintained port. Vulnerabilities
in upstream Deck Shelves should also be reported to the
[upstream project](https://github.com/santojon/Deck-Shelves/security).
